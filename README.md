# Weather-Journal App Project

## Overview
This is an asynchronous web app that uses Web API and user data to dynamically update the UI. It is a part of the nanodegree program that I am pursuing on Udacity.

## Technologies Used
<ul>
  <li>HTML5/CSS3</li>
  <li>JavaScript</li>
  <li>Node.js</li>
  <li>Express.js</li>
  <li>Openweathermap API</li>
</ul>
